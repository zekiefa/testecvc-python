- Run the project: 

### About the application
It's a RESTFull API for booking and searching a hotel.

### Access the application
http://localhost:8080/booking/9626/2019-09-24/2019-09-30/1/0, where:
- 9626: city code
- 2019-09-24: checkin
- 2019-09-30: checkout
- 1: adults
- 0: child

http://localhost:8080/hotel/6, where:
- 6: hotel ID
